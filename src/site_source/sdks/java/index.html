Getting Started with Java SDKs

Rackspace's official SDK for Java Apache jclouds, an open source multi-cloud toolkit for the Java platform that gives you the freedom to create applications that are portable across clouds while giving you full control to use cloud-specific features.

Currently, jclouds has support for multiple Rackspace services: Cloud Servers, Cloud Files, Cloud Block Storage, Cloud DNS, Cloud Databases, Auto Scale, Cloud Images, Cloud Load Balancers. Support for additional services is on-going.

Installation

The recommended way to install jclouds is to use Apache Maven. If you do not have a pom.xml file, you can copy and paste the one below. If your project already has a pom.xml file, just add the dependency section below into it.

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
  <modelVersion>4.0.0</modelVersion>
  <properties>
    <jclouds.version>1.7.3</jclouds.version>
  </properties>
  <groupId>com.mycompany.app</groupId>
  <artifactId>my-app</artifactId>
  <version>1.0-SNAPSHOT</version>
  <dependencies>
    <dependency>
      <groupId>org.apache.jclouds</groupId>
      <artifactId>jclouds-all</artifactId>
      <version>${jclouds.version}</version>
    </dependency>
  </dependencies>
</project>

Using jclouds

The jclouds Compute API provides both a basic abstraction across Compute APIs and lower abstraction layers, such as provider-specific APIs.

In this example, we create an API that can access OpenStack Nova API (Cloud Servers compatible) and then get the Server API for the IAD region. Then we get a set of Resources that contain server IDs.

  NovaApi novaApi = ContextBuilder.newBuilder("rackspace-cloudservers-us")
          .credentials("myUsername", "myApiKey")
          .buildApi(NovaApi.class);

  ServerApi serverApi = novaApi.getServerApiForZone("IAD");
  Set<Resource> servers = serverApi.listServers();
