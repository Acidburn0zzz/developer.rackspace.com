Getting Started with the Java SDK

Rackspace's official SDK for Java is Apache jclouds, an open source multi-cloud toolkit for the Java platform that gives you the freedom to create applications that are portable across clouds while giving you full control to use cloud-specific features.

Currently, jclouds has support for many Rackspace services:

  • Cloud Servers
  • Cloud Files
  • Cloud Block Storage
  • Cloud DNS
  • Cloud Databases
  • Auto Scale
  • Cloud Images
  • Cloud Load Balancers

Support for additional services is on-going.

Requirements

  1. Java Development Kit (JDK) version 6 or higher
  2. Maven version 3 or later

Installation

Download this pom.xml file or put the dependencies section in your project's pom.xml. Run the Maven command below in the same directory as your pom.xml file.

{% highlight sh %} $ mvn dependency:copy-dependencies{% endhighlight %}

Using jclouds

In this example, we build a NovaApi that can access Cloud Servers (which is based on the OpenStack Nova API) and then get the ServerApi for the IAD region. Then we get a List of Servers in that region.

{% highlight java %} NovaApi novaApi = ContextBuilder.newBuilder("rackspace-cloudservers-us") .credentials("myUsername", "myApiKey") .buildApi(NovaApi.class); ServerApi serverApi = novaApi.getServerApiForZone("IAD"); List servers = serverApi.listInDetail().concat().toList();{% endhighlight %}

See the Getting Started for Cloud Servers guide.
