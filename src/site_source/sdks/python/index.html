Getting Started with the Python SDK

Rackspace's official SDK for Python is pyrax, an Apache licensed SDK for OpenStack clouds.

A unified OpenStack SDK for Python is in the works.

Installation

Generally, pyrax should be installed into a virtual environment using pip

{% highlight sh %} $ pip install pyrax {% endhighlight %}

Getting started with Pyrax

{% highlight python %} import pyrax pyrax.set_setting("identity_type", "rackspace") pyrax.set_default_region('DFW') pyrax.set_credentials('{username}', '{apiKey}') {% endhighlight %}
